# Raspberry Pi

| Default User | Password |
|--------------|----------|
| rhino        | 1234     |

To install, simply flash the image to an SD card or NVMe drive, and plug it into the device to boot. You may need to run `unxz` on the image first to ensure it is not flashed with corruption.
